# Financial-Bank-Loan
Financial Loan Analysis Dashboard
# 📌 Project Overview
This Power BI project provides a comprehensive analysis of financial loan data to help stakeholders understand lending patterns, borrower profiles, and risk factors.
The dashboard transforms raw financial data into actionable insights regarding loan approvals, repayment statuses, and regional distribution. 


# 🚀 Key Features

Executive Summary: High-level KPIs including Total Loan Applications, Total Funded Amount, and Total Amount Received. 



Borrower Demographics: Analysis of loan distribution based on employment length, home ownership, and debt-to-income (DTI) ratios. 



Loan Performance Tracking: Visualization of "Good Loans" vs. "Bad Loans" based on repayment status. 


Temporal Analysis: Monthly trends (MTD) and year-over-year growth in lending activities. 



Regional Insights: Geospatial visualization of loan applications across different states and regions. 


# 🛠️ Tech Stack & Skills

Power BI Desktop: For report authoring and dashboard creation. 


DAX (Data Analysis Expressions): Used for creating complex measures like Year-to-Date (YTD) totals and custom calculated columns. 


Power Query: For data cleaning, transformation, and ETL processes. 


Data Modeling: Implementation of a star schema to optimize report performance and SQL

# 📊 Dashboard Pages

Summary Page: Focused on high-level KPIs and critical lending metrics. 


Overview Page: Provides detailed breakdowns by loan grade, sub-grade, and purpose. 


Details Page: A granular view of the dataset for deep-dive analysis into individual borrower records. 

# 📖 How to Use
Download the File: Download the financial_loan.pbix file from this repository.

Open in Power BI: Ensure you have the latest version of Power BI Desktop installed.

Interact: Use the slicers on the left-hand side of the report to filter data by State, Grade, or Employment Period.
